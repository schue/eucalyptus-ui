Manage Scaling Group Instances

This screen allows you to set the health status on the instances in your scaling group.

Searching and Filtering the Launch Configurations List

  1. Type some search text into the Search text box to filter the list of instances.
  2. For some lists, you can select one or more filters by pressing the enter key or clicking on the search bar and selecting an available filter from the drop down list. Selecting some filters will expose a menu with a list of valid filter values that you can select by pressing the enter key or clicking the value.

Setting the Health Status of an Instance

Select the health status for the instance with the Health Status drop-down list box.

Removing an Instance

Remove an instance from the scaling group by clicking the red minus sign to the right of the instance in the list.

Undeleting an Instance

You can undo a delete operation on an instance by clicking the grey x button to the right of the instance in the list.

Save Changes

To finalize the changes to your scaling group instances, click the Save changes button.
