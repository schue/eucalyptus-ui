Create a Scaling Group

This dialog allows you to create a scaling group. A scaling group is used to set up auto scaling for your cloud.

General

This section is where you specify the basic configuration of your new auto scaling group.

  1. Type the name of your new auto scaling group in the Name textbox.
  2. Select a launch configuration group from the Launch Configuration drop-down list box.
  3. Select the maximum number of instances you want running in your autoscaling group using the Maximum control.
  4. Specify the minimum number of instance you want running using the Minimum control.
  5. Specify the desired number of instance you want running using the Desired control.

Membership

In this section, you can define load balancers, availability zones, and health checks for the instances that run in your auto scaling group.

  1. Select the type of health check for your auto scaling group from the Type drop-down list box.
  2. Type the amount of grace period, in seconds, into the Grace period text box.
  3. Select the availability zone for your new scaling group from the Availability zones drop-down list box. You can add another availability zone to the new auto scaling group by clicking the plus button next to the drop-down list box.
  4. Select one or more load balancers for this auto scaling group from the Load balancers drop-down list box. You can add another load balancer to the new auto scaling group by clicking the plus button next to the drop-down list box.

Policies

In this section, you can define the scaling policies for your auto scaling group.

  1. Type a name for the scaling policy in the Name text box.
  2. Select an action type using the Action control.
  3. Type the numerical unit for the scaling action in the Amount text box. This value is used in conjunction with the value entered in the Measure control to determine how many instances to scale for this policy.
  4. Select the measure for the scaling action using the Measure control.
  5. Select a CloudWatch alarm from the Alarm control. You can click the Create Alarm link to display the Create Alarm dialog.

Saving the Scaling Group

Once you're satisfied with the properties you've defined for your scaling group, click the Create scaling group button to create your scaling group.
